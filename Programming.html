Programming and Software Development

Luca Monaco, Programmer and Software Developer

The C language, created in 1972 at Bell Laboratories, is currently the most used programming language in university introductory computer science courses, thanks to its diffusion in the professional field, the close link with the UNIX world and the ease with which it allows switching to other languages such as C++, Java and Perl.
C is a programming language that does not have a graphical interface: it is said to be a programming language with a command line interface. To use it, therefore, it is necessary to have a terminal: on Mac this is called Terminal, and it is an application that you will find in the Launchpad; on Windows instead there is both the Command Prompt and Microsoft PowerShell.
